# com-tty

`com-tty` 是一个基于 **Vue 3 + Vite + TypeScript** 的个人前端练习项目，聚焦于组件化开发、可视化展示与工程化实践。

项目集成了 Element Plus、Pinia、Vue Router、Three.js、ECharts、Mock 数据、WebSocket 管理等常见能力，用于探索从页面搭建到交互实现、从开发规范到构建发布的完整前端开发流程。

## 项目特点

- **组件化与模块化**：按业务拆分组件，支持全局组件注册与按需组织。
- **可视化能力**：包含图表展示与 3D 相关能力（如 Three.js / shader 相关模块）。
- **工程化完善**：内置 ESLint、Prettier、Commitlint、Husky、Vitest 等质量保障工具。
- **开发体验友好**：支持本地 Mock、主题能力、常见工具函数沉淀，便于快速迭代。

## 适用场景

适合作为中小型前端项目模板、个人技术实验场，或 Vue3 技术栈学习与工程实践参考。
